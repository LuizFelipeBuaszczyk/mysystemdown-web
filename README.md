# WEB

O projeto WEB serve como frontend web para o projeto MySystemDown, realizando integração com o (backend)[https://github.com/mysystemdown/backend].

## Tecnologias
As tecnologias utilizadas neste projeto são:

- Next.js
- Typescript

## Como Rodar

O servidor de desenvolvimento fica disponível na porta 3000. Para rodar o servidor de desenvolvimento:

```bash
npm run dev
# or
yarn dev
```
